# Contributing to Screen Translator 🤝

First off, thank you for considering contributing to Screen Translator! It's people like you that make open-source such a great community to learn, inspire, and create.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** locally with a descriptive name (`git checkout -b feature/awesome-new-feature`).
4.  **Commit changes** to the branch (`git commit -am 'Add some awesome new feature'`).
5.  **Push** to your fork (`git push origin feature/awesome-new-feature`).
6.  **Open a Pull Request** in our repository.

## What you can help with

-   **Adding more OCR Language support**: Currently supports Japanese, Korean, and Chinese via Google ML Kit.
-   **Improving the UI/UX**: The Settings page (`MainActivity`) and the translation bubble (`OverlayManager`) can always be prettier!
-   **Performance optimizations**: Making the screen capture process even more efficient.
-   **Bug fixes**: Squashing any bugs you encounter.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms. Let's keep it friendly and inclusive!

Happy coding! 🚀
